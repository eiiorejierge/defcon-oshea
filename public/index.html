    Changelog

    Version 1.2.0 (July 2026)

    • NEW Giphy GIF Search & Picker: Added a built-in GIF search button inside the chat input. Click to open trending GIFs, search Giphy in real time, and click to send.
    • NEW 30 FPS Screen Sharing: Increased screen sharing frame rate from 5fps to 30fps for ultra-smooth video streaming in the Voice Lounge.
    • IMPROVED Side-by-Side Split View: Relocated the stream presentation viewport to dock side-by-side with the chat screen (the chat remains 100% visible and interactive while watching).
    • NEW Theatre Mode & Fullscreen: Added expand buttons to collapse the chat and view the stream in focus layout, or use native fullscreen.
    • NEW Watch Roster Controls: Streams no longer auto-launch; instead, click the "Watch" button next to a streamer's name in the voice lounge roster list.
    • FIX Stale User Count: Switched Pusher authorization identifiers to username mapping and added explicit socket disconnection on tab close to resolve ghost/duplicate user listings.

    Version 1.1.0

    • NEW Agora Voice Lounge: Join voice lounge calls in Aether lounge channels directly.
    • NEW Discord Webhooks: Configure server-side alerts to announce user joins on external Discord channels.
    • IMPROVED Glassmorphic Dark Interface: Reworked typography, smooth transitions, and glass backdrops.
